# 坦克打字游戏 (Tank Typing Game)

一款适合儿童的打字练习游戏，通过二战坦克主题让打字学习变得有趣。

## 功能特点

- **多用户支持**：支持创建多个用户，每个用户独立保存学习进度
- **系统化训练**：56个训练课程，从单键练习到综合练习
- **坦克挑战**：22个坦克关卡，涵盖德国、美国、英国、苏联、意大利、日本等国家的著名坦克
- **难度递进**：从基础行到上排键、下排键，再到标点符号，循序渐进
- **进度保存**：使用localStorage自动保存学习进度
- **手指提示**：每个课程都有手指位置提示，帮助正确指法

## 游戏内容

### 训练课程（56课）
- 第1-5课：左手基础行（A S D F）
- 第6-11课：左手组合练习
- 第12-15课：右手基础行（J K L ;）
- 第16-19课：右手组合练习
- 第20-25课：左右手配合
- 第26-35课：上排键（QWERTYUIOP）
- 第36-38课：上排组合练习
- 第39-48课：下排键（ZXCVBNM）
- 第49-52课：标点符号
- 第53-56课：综合练习

### 坦克挑战（22关）
- 德国坦克：Panzer I、Panzer II、Panzer III、Panzer IV、StuG III、Tiger I、Tiger II、Panther、Jagdtiger、Jagdpanther
- 美国坦克：M3 Stuart、M4 Sherman、M26 Pershing、M18 Hellcat
- 英国坦克：Matilda II、Churchill、Cromwell、Firefly
- 苏联坦克：T-34、KV-1、IS-2、SU-100
- 意大利坦克：M13/40、P40
- 日本坦克：Type 97 Chi-Ha、Type 3 Chi-Nu

## 使用方法

1. 打开 `index.html` 文件即可开始游戏
2. 首次使用时创建用户
3. 选择用户开始游戏
4. 按照屏幕提示输入单词
5. 完成关卡后自动保存进度
